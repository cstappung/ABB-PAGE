# Logo personalizado

Para usar tu propio logo, coloca un archivo llamado **`logo.png`** en esta carpeta (`assets/logo.png`).

La aplicación lo detectará automáticamente al cargar y reemplazará el placeholder "TV" del header.

## Requisitos
- Formato: PNG (con fondo transparente recomendado)
- Tamaño sugerido: 256×256 px o mayor, cuadrado o ligeramente apaisado
- Peso: menos de 200 KB

## Alternativa
También puedes subirlo desde la interfaz: click sobre el cuadrito del logo en la barra superior → seleccionar archivo. El logo subido por la UI se guarda en el navegador (localStorage). El archivo `logo.png` aquí tiene prioridad cuando no hay logo guardado en localStorage.
